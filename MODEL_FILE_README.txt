IMPORTANT: You must add vectorcam.onnx to this folder before deploying.

Download it from your Google Drive (VectorCam/vectorcam.onnx — the file you exported
in notebook 10_onnx_export.ipynb) and place it right next to index.html in this repo.

The app loads it with the path './vectorcam.onnx', so it MUST sit in the same folder
as index.html (the repo root). ~8.5 MB.

Delete this txt file once you've added the onnx.
